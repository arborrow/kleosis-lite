Kleosis
=======
This is the readme file for Kleosis, a student information system module for Drupal.

Prerequisites
-------------
* Drupal 7.x
* Moodle 1.9.x or 2.x

Installation
--------------
To install Kleosis, see the INSTALL.txt file.

Contributions
-------------
Kleosis is an open source project and is welcoming contributors to participate in its development.

More Information
----------------
See the Kleosis home site at: [http://www.kleosis.org](http://www.kleosis.org/ "Kleosis Website") 

Good luck and enjoy!
